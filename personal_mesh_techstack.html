Captures frames from camera, reads image files. Feeds into vision pipeline.

02
Data Processing Layer
🗣️
Whisper (OpenAI)
Speech-to-Text
Local open-source ASR model. Use whisper or faster-whisper for speed. Runs fully offline.
pip install openai-whisper
👁️
Ultralytics YOLO
Object Detection
YOLOv8/v11 for fast object detection in camera frames. Returns bounding boxes + labels.
pip install ultralytics
🖼️
LLaVA (via Ollama)
Vision-Language Model
Multimodal LLM for image captioning and visual Q&A. Run locally via Ollama server.
ollama pull llava
✂️
LangChain / NLTK
Text Chunking & Cleaning
RecursiveCharacterTextSplitter for chunking. NLTK or spaCy for sentence tokenization and noise removal.
pip install langchain nltk
🏷️
Python datetime / UUID
Timestamp & Session Tagging
Attach ISO-8601 timestamps, session IDs, and metadata dicts to every chunk before storage.
Built-in
03
Memory Storage Layer
🔢
Sentence-Transformers
Embedding Model
all-MiniLM-L6-v2 or all-mpnet-base-v2. Generates dense 384/768-dim vectors locally.
pip install sentence-transformers
🟣
ChromaDB
Vector Database
Persistent local vector store. Semantic search with cosine similarity. Built-in metadata filtering.
pip install chromadb
🗄️
SQLite + SQLAlchemy
Relational Database
File-based relational DB for structured metadata, user profiles, health metrics, and activity logs.
pip install sqlalchemy
📦
Pandas
Data Manipulation
DataFrames for health metric analysis, log aggregation, and feeding data into storage pipelines.
pip install pandas
04–05
Health Intelligence & Query Engine
❤️
Custom Python Classes
User Profile & Health Engine
Plain Python dataclasses or Pydantic models for Age, Weight, Goals, Calorie/Habit trackers, Lifestyle scoring.
pip install pydantic
📐
NumPy / SciPy
Scoring & Calculations
Statistical computations for lifestyle scoring, BMI calculations, trend analysis on health metrics.
pip install numpy scipy
🔍
ChromaDB Query API
Semantic Search
collection.query() with embedding vectors. Returns top-k similar memory chunks with distances.
chromadb
⚡
Context Fusion (Custom)
Recency + Priority Weighting
Custom re-ranking logic: recency decay, priority scores, behavioral pattern tagging. Pure Python + Pandas.
Custom Module
06
LLM Processing (RAG Pipeline)
🤖
Ollama
Local LLM Runtime
Runs models like Mistral-7B, LLaMA 3.2, Phi-3 locally. REST API for inference. No cloud needed.
ollama.ai
🦙
Mistral 7B / LLaMA 3.2
Recommended Mini-LLM
4-bit quantized models (Q4_K_M GGUF). ~4GB VRAM or RAM. Good balance of speed + intelligence.
ollama pull mistral
🔗
LangChain
RAG Orchestration
Chains for retrieval-augmented generation. RetrievalQA, ConversationalRetrievalChain with memory.
pip install langchain
📝
Jinja2 / f-strings
Prompt Templates
Dynamic prompt assembly injecting user context, memories, health data into structured LLM prompts.
pip install jinja2
07
User Interface / Output
🖥️
Streamlit
Dashboard & Chat UI
st.chat_message for chat UI, st.plotly_chart for analytics. Fast prototyping with Python-only codebase.
pip install streamlit
📊
Plotly / Altair
Analytics Charts
Interactive health metric visualizations, habit streaks, calorie trends, lifestyle score over time.
pip install plotly altair
📄
ReportLab / fpdf2
Weekly Summary Reports
Generates PDF weekly summaries of activity, habits, health metrics, and AI recommendations.
pip install fpdf2
09
Security & Privacy Layer
🔐
cryptography (Fernet)
Data Encryption at Rest
AES-128-CBC symmetric encryption for SQLite DB files and ChromaDB storage. Key derived via PBKDF2.
pip install cryptography
🔑
python-dotenv / keyring
Secrets Management
.env files for config, OS keyring for encryption key storage. Never hardcode secrets.
pip install python-dotenv keyring
🏠
Offline-First (Ollama + Local DBs)
Zero Cloud Dependency
All inference, storage, and UI run locally. No data ever leaves the machine by default.
Architecture Choice
